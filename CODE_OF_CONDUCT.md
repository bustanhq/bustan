# Code Of Conduct

## Our Standard

This project expects respectful, professional collaboration.

Participants should:

- assume good intent while being direct about technical issues
- welcome constructive review and disagreement
- focus criticism on ideas, code, and decisions rather than people
- be respectful of different experience levels, identities, and backgrounds
- help keep the project useful for maintainers, contributors, and adopters

Unacceptable behavior includes:

- harassment, discrimination, or intimidation
- personal attacks or insulting language
- publishing private information without permission
- deliberately disruptive or bad-faith participation
- sexualized language or unwanted sexual attention

## Scope

This code of conduct applies to project discussions, issues, pull requests, review comments, release discussions, and any other project-managed communication space.

## Reporting

To report conduct problems, contact the maintainer privately at `hello@bustan.dev`.

Please include enough context for the report to be actionable. Reports will be handled as confidentially as practical.

## Enforcement

The maintainer may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

Responses may include warnings, temporary restrictions, or permanent removal from project spaces depending on severity and pattern.
